[회원 관리 프로젝트]

@프론트 엔드(회원 정보 입력/ 로그인/ 회원정보 검색) - html/css/js/jQuery

*회원가입 (html페이지작성- 이름/핸폰/아이디/암호)

-<from>태그를 사용해서 4개의 정보를 입력받고 특정페이지로 전송하는 HTML 작성
-만약, 필수정보를 입력하지 않으면 감지해서 에러메시지를 출력하는 기능 필요

*회원로그인(아이디/암호 입력)

-특정이벤트를 감지해서 로그인창이 보이도록해줌.

-입력한 이이디/암호를 서버로 전송하는 역학(미입력시 에러처리)


*회원 탈퇴(아이디/암호 입력)

-버튼을 눌러서 탈퇴할 회원의 정보를 입력하는 창의 보이도록 해줌

-버튼을 눌러서 탈퇴한 회원의 정보를 입력
